Detailed Description

EasyUnit namespace. This is the namespace containing all easyunit classes. Test class containing all macros to do unit testing. A test object represents a test that will be executed. Once it has been executed, it reports all results in the testPartResult linked list.

A failure occurs when a test fails (condition is false). An error occurs when an exception is thrown during a test. A success occurs if a test succeed (condition is true).
