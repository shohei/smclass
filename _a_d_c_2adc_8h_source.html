39  /*Set start condition and edge according to mode:
40  0 - No start (this value should be used when clearing PDN to 0).
41  1 - Start conversion now.
42  2 - Start conversion when the edge selected by bit 27 occurs on the P2.10 / EINT0 / NMI pin.
43  3 - Start conversion when the edge selected by bit 27 occurs on the P1.27 / CLKOUT /
44  USB_OVRCRn / CAP0.1 pin.
45  4 - Start conversion when the edge selected by bit 27 occurs on MAT0.1. Note that this does
46  not require that the MAT0.1 function appear on a device pin.
47  5 - Start conversion when the edge selected by bit 27 occurs on MAT0.3. Note that it is not
48  possible to cause the MAT0.3 function to appear on a device pin.
49  6 - Start conversion when the edge selected by bit 27 occurs on MAT1.0. Note that this does
50  not require that the MAT1.0 function appear on a device pin.
51  7 - Start conversion when the edge selected by bit 27 occurs on MAT1.1. Note that this does
52  not require that the MAT1.1 function appear on a device pin.
53  When mode >= 2, conversion is triggered by edge:
54  0 - Rising edge
55  1 - Falling edge
56  */
